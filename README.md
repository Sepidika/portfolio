## 🌐 Deploy via GitHub Pages

1. Buka **Settings** → **Pages**
2. Source: `Deploy from branch` → `main` → `/ (root)`
3. Klik **Save**
4. Tunggu 1–2 menit → live di `https://username.github.io/cerkas-id`
